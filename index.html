סנכרון GitHub

חברו את הריפו שלכם ל-Base44. כל push ל-main יעדכן אוטומטית את ה-Preview.

Terminal 
git remote add origin https://github.com/YOUR_USER/YOUR_APP
git push -u origin main
2

יצירת MCP Server

MCP מאפשר ל-Claude Code לתקשר עם Base44. צרו תיקייה חדשה:

mcp-server/package.json 
{
  "name": "base44-mcp-server",
  "version": "1.0.0",
  "type": "module",
  "dependencies": {
    "@modelcontextprotocol/sdk": "^0.5.0",
    "dotenv": "^16.3.1"
  }
}
.mcp.json (project root) 
{
  "mcpServers": {
    "base44": {
      "command": "node",
      "args": ["mcp-server/index.js"]
    }
  }
}
3

הגדרת Dev Mode

הנקודה הקריטית - בדיקה בטוחה של environment variables שלא תקרוס ב-Production:

Safe Dev Mode Check 
// WRONG - will crash in Base44
const IS_DEV = import.meta.env.VITE_DEV_MODE === 'true';

// CORRECT - safe check
const IS_DEV = typeof import.meta !== 'undefined' &&
               import.meta.env &&
               import.meta.env.VITE_DEV_MODE === 'true';
4

עדכון Backend Functions

כל Function צריכה לתמוך ב-profile_id מהבקשה (ל-dev mode) בנוסף ל-auth.me():

Backwards-compatible Function Pattern 
let profile_id;

// Dev mode support
if (body.profile_id) {
    profile_id = body.profile_id;
} else {
    // Original production flow
    const user = await base44.auth.me();
    const profiles = await base44.asServiceRole
        .entities.Profile.filter({ user_email: user.email });
    profile_id = profiles[0].id;
}
